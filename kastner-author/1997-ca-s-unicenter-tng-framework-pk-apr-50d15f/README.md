## Abstract

Aberdeen Group analyst Peter S. Kastner profiles Computer Associates International's Unicenter TNG Framework, a free/bundled subset of Unicenter TNG announced July 14, 1997 at the CA World conference. The Framework provides auto-discovery, an object repository, 2D/3D mapping, event management, scheduling, and virus scanning—enabling 10-minute enterprise network discovery. Kastner analyzes CA's strategic shift to a 'razor/razor-blade' distribution model, bundling the Framework with server hardware from nine partners including HP and Digital Equipment, to establish it as a de facto enterprise management standard against IBM Tivoli, HP OpenView, and Sun Solstice.

## Key Observations

- CA Unicenter TNG ran at a **$1 billion annual run rate** with 7,000+ customers in FY1997
- Kastner's "razor/razor-blade" distribution model — give away Framework, sell full Unicenter — presaged modern freemium SaaS strategies by 15+ years
- Aberdeen installed Framework on its own NT server in **10 minutes**, discovering all enterprise devices automatically
- HP's own systems division endorsed Unicenter Framework as preferred management platform, undermining HP OpenView
