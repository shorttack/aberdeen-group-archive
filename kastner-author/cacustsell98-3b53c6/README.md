## Abstract

This document outlines the curriculum for a 'Customer Focused Selling' multi-day sales training program developed for Computer Associates' sales organization in 1998, covering behavioral style analysis using the DiSC model, trust-building techniques, prospecting and pre-call planning, customer-focused interviewing using the F.I.N.D. system, and competitive selling strategies. The program integrates personality profiling, active listening skills, and structured sales process methodology to equip CA sales professionals with consultative selling techniques suited to complex enterprise software deals.
