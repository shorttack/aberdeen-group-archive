# PSK-10 — frame-by-frame JPEG previews

These 12 JPEG files (`PSK-10-0.jpg` … `PSK-10-11.jpg`) are per-frame previews
extracted from the original `PSK-10.tiff` multi-frame TIFF (12 frames, 207 MB
in total). The original TIFF is held outside this git repository because it
exceeds GitHub's 100 MB per-file limit; these JPEG previews provide
sufficient archival reference for the collection.

See the parent directory's `c1990s-psk-photo-collection-misc.md` for the
full frame-by-frame index and identification notes.
