# DataFocus' NuTCRACKER: Enterprise-Class Unix Applications for NT

**Study ID:** 1997-datafocus'-nutcracker--enterprise-c-bfc11e

**Date:** September 1997  
**Type:** Executive White Paper  
**Subject:** DataFocus Incorporated — NuTCRACKER Unix-to-NT porting toolkit  

## Overview

Aberdeen Group executive white paper evaluating DataFocus' NuTCRACKER software development kits for porting enterprise-class Unix applications to Microsoft Windows NT. The study documents IS decision maker experiences, cost savings, performance benchmarks, and the strategic relationship between DataFocus and Microsoft. NuTCRACKER Version 3.1 (released May 1997) enabled companies to port Unix applications to NT in days, with cost recovery typically after the first server replacement. DataFocus was a private company with ~$7M revenue (1997) expected to double.
