## Overview

This is a brief Aberdeen Group vendor profile examining DataReach, a joint product developed by EMC Corporation and BMC Software to solve the problem of mainframe data extraction for open systems environments. The abstract-level profile (approximately 1,450 characters of extracted text) argues that DataReach would "topple mainframe-infrastructure barriers" and help large enterprises capture business value from their mainframe data assets.

Aberdeen predicted that DataReach — combining EMC's intelligent storage hardware with BMC's data movement software — would bypass mainframe scheduling constraints and speed delivery of critical data to data warehouses and data marts.

## Notes

This study is notably brief — only the abstract page was captured (1,450 chars), as the full text required authentication on the Aberdeen website. DataReach as a joint EMC-BMC product did not achieve long-term market traction; EMC was acquired by Dell in 2016, and the mainframe data extraction market evolved toward ETL platforms and cloud data integration tools.
