**Study ID:** 1997-dec-linkworks-webworker---an-object-db2d2a

**Date:** April 1997  
**Type:** Vendor Profile  
**Subject:** Digital Equipment Corporation LinkWorks WebWorker  

## Overview

Aberdeen Group profile of Digital Equipment Corporation's LinkWorks WebWorker product — a distributed object framework enabling enterprise document management and workflow to extend to the Web. The study evaluates LinkWorks' three years of distributed object deployment experience against competitors still planning object frameworks, and recommends evaluation of LinkWorks for enterprise IS extroversion strategies.
