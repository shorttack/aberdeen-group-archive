## Overview

This Aberdeen Group strategy profile evaluates Business Objects — the company, its BusinessObjects 4.0 query/reporting/analysis platform, and its installed base — as a top-tier multi-product software company with solid financial performance and a strong pattern of innovation. Written in early-mid 1997, the profile covers:

- BusinessObjects 4.0 architecture: integrated Reporter, Explorer, Document Agent Server, Designer, Supervisor modules
- The Microcube engine for client-side OLAP without pre-built cubes
- Project Darwin: Business Objects' planned web/internet three-tier application server
- BusinessMiner: desktop data mining tool (decision-tree technology licensed from ISoft)
- Financial health: $85.1M in 1996 revenue, $40M cash, zero debt, 40% YoY growth
- 475,000 licenses, 4,000 enterprises across 60 countries
- SAP designation of BusinessObjects 4.0 as sole decision-support tool in Complementary Software Partner Program

Key Aberdeen predictions:
- Business Objects will sustain healthy growth and market leadership
- Project Darwin will reduce total cost of ownership for BI software
- BusinessMiner will pressure high-end data mining software suppliers

## Notes

Business Objects was acquired by SAP for $6.8 billion in January 2008, making it the largest acquisition in SAP's history at the time. The company grew from $85M in 1996 revenues to $1.5B by end of 2007. Project Darwin became WebIntelligence. BusinessMiner was later discontinued. The SAP alliance mentioned in this profile ultimately led to the acquisition.
