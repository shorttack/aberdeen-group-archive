## Overview

Full-text Aberdeen Group profile of Prism Solutions Inc. (Sunnyvale, CA), evaluating its metadata-driven data warehouse software suite. Aberdeen endorses Prism's metadata-first architecture — Prism Warehouse Executive, Prism Warehouse Directory, Prism Iterations methodology — as the key to managing dynamic data warehouse evolution and scaling. The report covers CASE tool integrations, BI tool metadata access, CORBA/DCOM object architecture plans, Web Access and Personal Mart modules.

**Historical note:** Prism Solutions IPO'd in 1996 at ~$35/share, fell to ~$2 by 1998, and was acquired by Ardent Software for $42M in January 1999. The technology lineage: Prism → Ardent → Informix → Ascential Software → IBM InfoSphere DataStage.
