Abstract: Announcement Proﬁle
IBM Corporation — Enterprise Server Group
IBM Ships First 64-bit RS/6000 Server
On October 6th IBM announced availability of the newest
member of its RS/6000 product family, code named Raven.
This S70 model is a 64-bit, 4-to-12 way server that
incorporates PCI bus technology and delivers an estimated
18,000 Transactions Per Minute in TPC benchmarks.
Last year Aberdeen forecast the Spring 1997 roll-out of SMP
servers based on the PowerPC 604e chip -- which took place
on schedule with the roll-out of the RS/6000 F50 in May. The
second phase of 1997 product announcements revolved around
the introduction of a high-end SMP server using 64-bit
microprocessors and an enhanced version of AIX -- IBM’s
version of Unix. Now this phase of the upgrade program has
also been successfully completed.
The purpose of this Announcement Proﬁle is to review IBM’s
Unix market position and strategy as well as where the new
64-bit RS/6000 S-70 Raven model ﬁts in.
Tom Willmott
