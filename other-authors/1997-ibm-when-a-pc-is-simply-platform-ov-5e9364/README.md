## Overview

This data package archives Aberdeen Group's 1997 evaluation of **IBM's Network Station** network computer — one of the most detailed analyst assessments produced during the peak of the 1990s Network Computer (NC) movement.

The study evaluates:
- **Core technology**: PowerPC 403-based diskless appliance with terminal emulators, Java, and WinCenter Pro server access
- **Cost proposition**: $695 base NC vs. $2,000+ PC; $4,000-$15,000/year PC support cost savings
- **NC Reference Profile**: The IBM/Oracle/Sun/Apple/Netscape consortium standard (May 1996)
- **Use cases**: Multi-server access users; legacy terminal replacements; Java application environments

**Aberdeen's conclusion:** IS managers should evaluate the Network Station for appropriate deployments; it represents processing overkill avoidance.

## Predictions vs. Outcomes

| Prediction (1997) | Outcome | Result |
|---|---|---|
| NC deployment will reduce desktop costs | NC failed; Chromebooks/VDI confirmed the model 14+ years later | Delayed Vindication |
| Java broadly accepted as primary dev environment | Server Java dominant; desktop Java failed; JS dominates browser | Partial |
| Current OS (Win95/NT) obsoleted within 18 months | Transition happened but more slowly than predicted | Overstated |
| NC viable alternative to PC for IS-managed orgs | Chromebooks (2011+) and VDI/Cloud PC confirmed the concept | Ultimately Confirmed |
| NC market will develop from 5-company alliance | Alliance fragmented; NC brand abandoned by 2000 | Not Confirmed |
