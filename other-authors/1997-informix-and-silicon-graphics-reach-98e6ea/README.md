## Abstract

Aberdeen Group analyzes the joint TPC-C benchmark result achieved by Informix Software and Silicon Graphics. The Informix-OnLine Dynamic Server 7.3 running on the Silicon Graphics Origin2000 server (28-way SMP, 195-MHz MIPS R10000 CPUs, 13 GB RAM) achieved 25,309.20 tpmC — the highest non-clustered SMP result at the time and second overall. The $139.04/tpmC price-performance was best-in-class for $3M+ systems. Aberdeen argues this signals SGI's serious entry into commercial OLTP, validates 64-bit VLM architectures for database workloads, and marks Informix's return to benchmark leadership after a period of low visibility.
