## Abstract

Aberdeen Group evaluates Traveling Software's LapLink remote control solutions as a telecommuting enablement platform. The study highlights LapLink's role in addressing the telecommuting challenge — enabling remote workers to control host computers over dial-up connections, synchronize files, and access corporate network applications with near real-time performance. Aberdeen notes that over 9.5 million employees work off-site in 1997, projected to exceed 15 million by 2000, and identifies LapLink as a strong solution for IT organizations supporting telecommuters. Key features include SpeedSync file synchronization (80% reduction in transfer times), security controls, and a remote control architecture that outperforms remote node for bandwidth-intensive applications.

## Context & Outcome

Traveling Software (later renamed LapLink Software) survived and remains active as of 2026, though it pivoted from remote control to PC migration software. The company's PCmover product became its flagship. Aberdeen's prediction of telecommuting growth was correct — Gallup data shows telecommuting grew from 9% in 1995 to 37% by 2015, and post-COVID remote work became near-universal. However, LapLink's specific remote control approach was largely superseded by VPN solutions, Microsoft Remote Desktop Protocol (RDP), and later cloud-based tools like Citrix, LogMeIn, and Teams. The telecommuting market Aberdeen identified was correct; the technology vehicle (dial-up remote control) was a transitional solution.
