The NCR 3550 is a tightly-
coupled, fully symmetric multipro­
cessing system linking two to eight
5OMHz Intel486™ processors,
each operating at 40 MIPs for a
total of 320 MIPs. Each processor
in the symmetrical architecture is
supported by 256K of cache, can
perform any function, and can take
full advantage of multiprocessing
capabilities.

The tighdy-coupled, symmetric
architecture of the NCR 3550
provides compatibility with System
3000 uniprocessing applications;
there is generally no need to modify
applications to run on this multi­
processing system.

 NCR UNIX System V Release 4
 Operating System
 The NCR UNIX System V Release
 4 (SVR4) operating system is
 designed to offer a significant
 competitive edge. This operating
 system scales the entire System
 3000 family taking full advantage of
 both uniprocessor and multiproces­
 sor hardware designs. The NCR
 UNIX System V Release 4 operat­
ing system is a fully symmetrical
implementation for multiprocessing
systems such as the NCR 3550.
This allows any processor to
execute any combination of kernel
and user processes or I/O requests,
virtually eliminating potential
software bottlenecks. In addition,
the kernel is multi-threaded,
meaning that multiple parts of it
can be executed in parallel.

Based on requirements defined by
UNIX International, SVR4 com­
bines the functionality of UNIX
